Web Crawler and Search Engine

Web Search and Information Retrieval (COEN 272)

Part 1

  • Goal: Implement a simple web crawler in Java that saves HTML files to a local document store and perform noise removal and term analysis
  • Saved files were cleaned for noise (i.e. ads, extraneous links) using the JSoup library for HTML/DOM manipulation
  • Zipf's Law analysis was then performed on the cleaned HTML files

Part 2

  • Goal: Using a web crawler from Part 1, compile a store of HTML pages to access via a search engine
  • Our team used Apache Solr to perform the back-end retrieval of results for the search engine result page
  • Personal responsibilities included: HTML file parsing, data serialization, and technical documentation
  • We then performed A/B testing and analysis with emoji search as a novel feature for housing search
